# ⚛️ Next.js Dashboard

**Language:** TypeScript | **Role:** Unified Monitoring UI

Real-time dashboard for the Supply Chain Risk Intelligence System. Displays live risk events, supplier graph visualisations, and mitigation plans generated by the AI pipeline.

---

## Directory Structure

```
frontend/
├── package.json                    # Next.js app config
├── next.config.ts                  # Next.js config
├── tsconfig.json                   # TypeScript config
├── public/
│   └── images/                     # Static UI assets
└── src/
    └── app/
        ├── layout.tsx              # Root layout
        ├── globals.css             # Base styles
        ├── page.tsx                # Landing page (Home)
        ├── architecture/           # Infrastructure & systems overview
        ├── console/                # Tactical operations console route
        ├── devdocs/                # Developer API & console logs
        ├── ecosystem/              # Agent topology & supplier graph
        └── intelligence/           # Live intelligence dashboard
```

---

## Running

```bash
cd frontend
npm install
npm run dev       # starts on http://localhost:3000
npm run lint      # runs ESLint
npm run build     # creates a production build
```
